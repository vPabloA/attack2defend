# Navigator UI

React + Vite + TypeScript MVP UI for Attack2Defend.

## UX priority
1. Route first
2. Graph second
3. MITRE native views third

## Runtime data
- Primary: `/data/knowledge-bundle.json` (generated by knowledge builder)
- Fallback: `src/data/log4shell.route.json`

## Tabs
- Route
- Actions
- Graph
- MITRE Views
- Coverage
- Export

## Local run
```bash
npm install
npm run build
npm run dev
```
